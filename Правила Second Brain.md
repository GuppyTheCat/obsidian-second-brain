# Правила Second Brain

Этот документ описывает правила и структуру моей базы знаний в Obsidian.

## Правила

- **Минимум папок** — заметки организуются через свойство `categories` и `tags`, а не через файловую структуру
- **Свойства в нижнем регистре** — все frontmatter-свойства пишутся с маленькой буквы
- **Ссылки вместо папок** — активное использование `[[внутренних ссылок]]` для связывания заметок
- **Bases для навигации** — `.base` файлы создают табличные представления заметок по категориям
- **Форматирование дат** - везде использовать формат `YYYY-MM-DD`
- **Шкала оценок** - описана [[Система оценок|здесь]] 

---

## Порядок свойств во frontmatter

- **Сверху (по порядку):** `categories` → `title` или `name` → `aliases` (если есть).
- **Середина:** остальные свойства (порядок между ними не задан).
- **Снизу (по порядку):** `published` (если есть) → `created` → `tags`.

## Структура

| Папка | Назначение |
|-------|------------|
| `/` (корень) | Заметки |
| `.cursor/`    | Конфигурация для Cursor, в том числе Cursor Rules и Agent Skills |
| `.obsidian/`   | Конфигурация для Obsidian |
| `Attachments/` | Изображения, аудио, видео и другие вложения |
| `Bases/` | `.base` файлы для Obsidian Bases |
| `Categories/` | Обзорные страницы категорий |
| `Clippings/` | Сохранённые материалы из интернета (посты, статьи, страницы, видео) |
| `Daily/` | Ежедневные заметки в формате `YYYY-MM-DD.md` |
| `Files/Books/` | Файлы книг |
| `Files/Excalidraw/` | Диаграммы и рисунки |
| `Files/Guitar Tabs/` | Гитарные табы |
| `Scripts` | Скрипты для создания представлений заметок и элементов интерфейса |
| `Templates/` | Шаблоны для новых заметок |
